hexo + github 搭建博客

hexo

默认你下载了git bash 和node.js

1.进入git bash
进入根目录，创建文件夹
cd /
mkdir blog

2.安装hexo
npm install -g hexo

3.初始化
hexo init

4.生成静态页面
hexo generate

5.修改配置文件
vim _config.yml
修改:
deploy:
type: git
repo: git@github.com:jiatianyao3/jiatianyao3.github.io.git
branch: master

修改账号
git config –global user.name jiatianyao3
git config –global user.email 258409707@qq.com

小问题：因为机器原先有github，想要用新的账户需要
新生成秘钥
ssh-keygen -t rsa -C “258409707@qq.com”
ssh-add ~/.ssh/id_rsa_work
如果报错，先
eval ssh-agent
再ssh-add id_rsa-work

在.ssh
目录创建文件
touch config
输入;
Host github.com
HostName github.com
User git
IdentityFile C:/Users/Administrator/.ssh/id_rsa

second user(second@mail.com)

建一个github别名，新建的帐号使用这个别名做克隆和更新

Host github2
HostName github.com
User git
IdentityFile C:/Users/Administrator/.ssh/id_rsa-work

添加完github账号就可以发布了
hexo d

启动
hexo server –config custom.yml

问题：
1.如果想直接用hexo server需要
npm install hexo-server –save

2.如果发现打不开网址，可能你的4000端口占用，需要换个端口
hexo server -p 5000
